## 💻 Website Features

This website is a modern, responsive single-page application built with:

### Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with CSS variables
- **JavaScript (ES6+)** - Interactive functionality
- **Font Awesome 6.4.0** - Icons
- **EmailJS** - Contact form email integration
- **Firebase** - Real-time database for packages and reviews
- **Google Fonts** - Typography

### Key Features

- 🏠 **Home Section** - Hero banner with call-to-action and feature highlights
- ℹ️ **About Section** - Company information and statistics
- 🎁 **Tour Packages** - Filterable tour packages by region (Europe, Asia, Africa, Americas, Oceania)
- ⭐ **Travel Reviews** - Customer reviews with star ratings and submission form
- 📬 **Contact Section** - Contact information and WhatsApp integration
- ⚙️ **Admin Panel** - Protected admin dashboard for package management
- 📱 **Fully Responsive** - Works on all devices
- 🌐 **WhatsApp Floating Button** - Quick contact via WhatsApp chat popup

## 🚀 Getting Started

### Prerequisites

- Any modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection (for loading external resources)

### Running Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/soin-tours.git
```

2. Navigate to the project directory:

```bash
cd soin-tours
```

3. Open `index.html` in your web browser:

```bash
# For Windows
start index.html

# For Mac
open index.html

# For Linux
xdg-open index.html
```

Or simply open the file directly in your browser by double-clicking it.

## 🔧 Customization

### Changing Colors

Edit the CSS variables in `styles.css`:

```css
:root {
  --primary-color: #3498db; /* Main brand color */
  --secondary-color: #2c3e50; /* Secondary color */
  --accent-color: #e74c3c; /* Accent/highlight color */
  /* ... other variables */
}
```

### Updating WhatsApp Number

The WhatsApp number is used in multiple places. Update in:

- `index.html` - Contact section and footer
- `main.js` - WhatsApp floating button functionality
